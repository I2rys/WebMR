## Installation
Github:

    git clone https://github.com/I2rys/WebMR

NPM Packages:

    npm i random-useragent && npm i request && npm i chalk && npm i delay
    
## Usage

    node index.js <url> <bots>

 - url - The url of the website to make a request on.
 - bots - The amount of bots to use.
